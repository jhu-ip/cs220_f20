      Course Objectives

      Course Objectives#

      Upon successful completion of this course, you are able to:

      1. Read, write, trace, test, and debug C and C++ program codes.
      2. Use command-line Unix/Linux tools for file management, creating and debugging programs in C/C++.
      3. Analyze a programming specification, design a program that fulfills the specification, create a development plan, execute the plan, implement and deliver the program on time.
      4. Utilize a distributed version control system to manage the development of a program.
      5. Understand the importance of good programming practices such as modularity, testing, documentation, and incremental design.
